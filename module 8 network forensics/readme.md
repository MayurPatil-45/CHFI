## Overview

This module focuses on **Network Forensics**, which involves capturing, analyzing, and investigating network traffic to detect security incidents, intrusions, malware communication, and data exfiltration. It helps investigators reconstruct network-based attacks and understand attacker behavior across enterprise networks.

---

## Learning Objectives

* Understand network communication and protocols from a forensic perspective
* Capture and analyze live and recorded network traffic
* Identify malicious traffic and attack patterns
* Perform packet, session, and flow analysis
* Reconstruct network-based incidents and timelines

---

## Topics Covered

### 1. Network Fundamentals for Forensics

* OSI & TCP/IP models
* Common network protocols (TCP, UDP, ICMP, HTTP, DNS)
* Ports and services

### 2. Traffic Acquisition

* Live packet capture
* Network taps and SPAN ports
* PCAP file analysis
* Evidence handling and integrity

### 3. Packet Analysis

* Packet structure and headers
* TCP three-way handshake
* Session reconstruction
* Protocol dissection

### 4. Protocol Analysis

* HTTP/HTTPS traffic analysis
* DNS query investigation
* Email protocols (SMTP, POP3, IMAP)
* FTP and SMB traffic

### 5. Malware & Attack Detection

* Command and Control (C2) traffic
* Beaconing behavior
* Suspicious payloads
* Data exfiltration techniques

### 6. Intrusion & Attack Investigation

* Brute force attacks
* DDoS traffic patterns
* Man-in-the-Middle attacks
* Lateral movement detection

### 7. Log & Flow Analysis

* Firewall logs
* IDS/IPS alerts
* NetFlow analysis
* Correlating logs with PCAP data

### 8. Wireless Network Forensics

* Wi-Fi traffic capture
* 802.11 frames
* Rogue access points
* Wireless attack detection

### 9. Timeline & Correlation

* Time-based traffic analysis
* Multi-source evidence correlation
* Incident reconstruction
