Overview

This module covers Linux and macOS Digital Forensics, focusing on the acquisition, analysis, and interpretation of forensic artifacts from Unix-based operating systems. It helps investigators analyze user activity, system logs, persistence mechanisms, and security incidents across Linux and macOS environments.

Learning Objectives

Understand Linux and macOS OS architecture from a forensic perspective

Identify key forensic artifacts in Linux and macOS systems

Perform live and dead forensic analysis

Analyze logs, user activity, and persistence techniques

Reconstruct attacker activity and timelines
