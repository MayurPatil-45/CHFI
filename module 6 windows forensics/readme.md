Overview

This module focuses on Windows Digital Forensics, covering the acquisition, analysis, and interpretation of forensic artifacts from Windows operating systems. It helps investigators understand user activity, system behavior, persistence mechanisms, and evidence reconstruction during incident response and forensic investigations.

Learning Objectives

Understand Windows OS architecture from a forensic perspective

Identify and analyze key Windows forensic artifacts

Perform live and dead system analysis

Reconstruct user actions and timelines

Detect persistence, malware traces, and attacker activity
