This module focuses on understanding how hard disks and file systems store, manage, and organize data at a low level. It provides essential knowledge for digital forensics, incident response, and cyber investigations, enabling investigators to analyze storage media effectively and identify hidden, deleted, or manipulated data.
Key Topics Covered

Basics of Hard Disk Architecture

Disk components: Platters, Tracks, Sectors, Clusters

Logical vs Physical Storage

Disk addressing methods (CHS, LBA)

File System Fundamentals

Common file systems:

FAT16 / FAT32

NTFS

EXT (EXT2 / EXT3 / EXT4)

File system structures:

Boot Sector

File Allocation Table / Master File Table (MFT)

Inodes and Metadata

File and directory organization

File attributes and permissions

Deleted files and data recovery concepts

File system timestamps (MAC Times)

Slack space and unallocated space

Tools & Techniques

Disk analysis tools

File system viewers

Hex editors

Forensic imaging tools

Data recovery utilities
Learning Outcomes

After completing this module, you will be able to:

Understand how data is stored on hard disks

Analyze different file systems during investigations

Identify deleted, hidden, or corrupted files

Interpret file metadata and timestamps

Support forensic investigations with disk-level analysis
